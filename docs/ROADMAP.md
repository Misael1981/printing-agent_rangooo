1. **Auto-Launcher (Obrigatório para Pizzaria)**: Dono de pizzaria não quer ter que lembrar de abrir o app. Podemos configurar para o Agente Rangooo iniciar sozinho assim que o Windows ligar.

2. **Telinha de Configuração de Token**: Em vez de você ter que editar o .env e gerar um novo .exe para cada cliente, podemos criar um botão de "Configurações" ⚙️ onde ele mesmo cola o token dele.

3. **Logs Persistentes**: Atualmente, se ele fechar o app, o log some. Podemos fazer o sistema salvar um arquivo log.txt na pasta do cliente para você poder dar suporte remoto se algo falhar.
